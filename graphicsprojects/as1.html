Shading

My partner and I created a program that simulates different lights illumniating a sphere.

an image showing diffuse only shading from a single light source :

command : ./as1 -kd .78 .08 .52 -dl -1 -1 -1 1 1 1

an image showing specular only shading from a single directional light source :

command : ./as1 -ks .416 .35 .8 -dl 1 -1 1 1 1 1

an image showing specular only shading from a single point light source :

command : ./as1 -ks 0 .74 1 -pl 1 1 1 1 1 1

an image showing combined specular and diffuse shading :

command : ./as1 -kd .7 .5 .2 -ks .2 1 1 -sp 40 -pl 1 1 1 .21 .6 .9 -ka .4 .2 .5 -dl 1 1 1 .3 .3 .3

an image showing anisotropic specularity specularity :

command : ./as1 -kd .86 .62 86 -ks .28 .82 .8 -sp 2 -dl -1 -1 -1 1 1 1 -pl 1 1 -1 -1 1 1 -spu 50 -spv 10

an image with multiple light sources :

command : ./as1 -ks .69 .87 .9 -dl -1 -1 -1 1 1 1 -sp 2 -pl 5 -1 2 1 .62 .47 -dl -3 8 9 0.29 0 .18
